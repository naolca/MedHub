# MedHub
MedHub is a one-stop solution for all your pharmacy needs. It is a user-friendly application that helps you manage your pharmacy, keep track of your inventory, and also allows you to search for medicines at different pharmacies.

## Features
### As a Pharmacy Management System:
- Keep track of your inventory.
- Manage sales and purchase orders.
- Generate reports and analytics.
### As a Medicine Search Engine:
- Search for medicines at different pharmacies.
- Find the availability of specific medicines and in which nearby pharmacy they can be found.

## Installation
To install and run MedHub, follow these steps:
1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Install the required dependencies.
4. Run the application.
